osltxt-format

A minimalist plain-text file format with open-source licensing and cryptographic integrity protection (SHA-256).

Installation via Scoop

If you use the Scoop package manager, you can install osltxt by adding your custom bucket and running the install command:

scoop bucket add scoop-osltxt https://github.com/socialmediatvtweet-tech/scoop-osltxt

scoop install osltxt
